PeerBlock Quick Start Wizard

When PeerBlock opens for the first time, the "PeerBlock Startup Wizard" will appear. This wizard will help you choose some of the standard settings to make PeerBlock work most effectively for you!

  1. To continue with the Startup Wizard, click "Next"

    Click to see full size
  2. This window gives you two options regarding block/allow lists. These options are:
    1. Which standard lists you wish to use, with a brief description. For more details on which lists to choose, see Which lists should I use?. If you wish to add more lists, select "Import\Create Lists". The next step (which is identical to "List Manager", see List Manager) will allow you to do this, however it is not shown here. For standard use, the default options("P2P" checked only) are sufficient.
    2. This option will allow connections on "Port 80" and "Port 443", which are standard ports used to browse the web. Whilst this will allow you to browse the entire web without interuption, "bad" connections can also go through these ports. To increase safety, it is recommended to leave this unchecked, and manually add websites of interest later if there are any issues viewing particular websites.
    Once you have finished, click "Next" to continue.

    Click to see full size
  3. This window also has two options, this time regarding updates. The options are:
    1. The frequency of list updates. The more frequently they are updated, the better the protection, however increasing the update frequency also increases the bandwidth (download) usage. Typically, the default option ("Every other day") will suit most users.
    2. Which updates will be checked automatically. If you select "PeerBlock", it will check for an updated version, and if found will prompt you to visit the website to update to the newest version. Choosing to update "Lists" will keep your block lists more accurate, and increasde protection. It is recommended that "PeerBlock" be checked, and strongly recommeneded that automatic list updates are checked.
    Click "Next" when you have finished.

    Click to see full size
  4. Click "Finish" to start PeerBlock. The update process will complete, then PeerBlock is ready to use!
