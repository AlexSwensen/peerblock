Platform Support
---------------------
PeerGuardian 2 currently supports Windows 98, ME, 2000, XP, 2003, Vista,
and 2008 in 32-bit and 64-bit.

Source Code
-------------------
PeerGuardian 2 is available freely under a zlib License (license.txt).
Please make sure you understand it well before messing with the included
source.  Though not required, we would love to hear from you at the Methlabs
forums if you use it.  Read compiling.txt for instructions to build.
