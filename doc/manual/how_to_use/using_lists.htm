Using Lists in PeerBlock

Once you have decided which lists you would like to use in PeerBlock, you need to set PeerBlock up to use these lists. If you have not already chosen the lists you would like to use, see Selecting Appropriate Lists

To use lists in PeerBlock, follow these steps:

  1. Open PeerBlock, and select "List Manager"

    Click to see full size
  2. This window show all of the cureent lists in use by PeerBlock. To add another list, either select from the default lists, or click "Add" to add a different list.

    Click to see full size
  3. Here, you add the details PeerBlock required to use your lists. the information required is:
    1. A brief description to help you remember the purpose of the list.
    2. The location of the list. Typically, list your will use will be from the internet, so enter the "URL" of the list here.
    3. You need to choose if you would like PeerBlock to block IP's on this list, or allow them if they are blocked on other lists. In most cases, you will use "Block".
    4. Once you've finished adding the details, click "OK".


    Click to see full size
  4. If you would like to add more lists, comlete steps 2 and 3 again. otherwise, click the close button in the top right of this window.

    Click to see full size
  5. PeerBlock will automoatically download these lists so they are ready to use. Once this completes, your new lists are ready to use! From here, you may like to learn about HTTP Blocking or How and why to use allow lists.
